# First Steps with AI Tools

A 90-minute conference talk introducing teachers to working with AI tools. Delivered on 8 May 2026 at KSC's AI in Education conference.

The talk has four sections: what generative AI actually is, prompting for reliable results, AI for classroom content, and students using AI in assignments.

You can access the presentation by clicking [here](https://github.com/EdinGuso/First-Steps-with-AI-Tools/blob/main/presentation.pdf), and the accompanying prompt patterns handout by clicking [here](https://github.com/EdinGuso/First-Steps-with-AI-Tools/blob/main/prompt_patterns_handout.pdf).
